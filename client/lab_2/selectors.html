First, the basics.

You can style unordered lists with the following bullets:

  • disc
  • circle
  • square

And you can number ordered lists with the following:

  1. decimal
  2. lower-roman
  3. upper-roman
  4. lower-alpha
  5. upper-alpha
  6. (and many more!)
CSS selectors let you select individual HTML elements in an HTML document. This is super useful. Classes are ridiculously important, since they allow you to select arbitrary boxes in your web pages.
